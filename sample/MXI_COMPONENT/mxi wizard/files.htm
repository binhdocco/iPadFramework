files=Files source

The name of the file. For this version of MXI Wizard all the extension's files MUST be in the same folder. Filenames should be a total of 30 characters or less.
Click the "%2B" button to add files and "-" to remove.
Then select the file in the list, enter the relevant attributes and click "Assign Values" button.
NOTE: If you do not click "Assign Values" button the attributes will not be assigned to the file.

Destination

The name of the folder the file will be copied to. If the folder doesn't exist, the Extension Manager creates it during installation. Note that this attribute should contain a folder name, not a filename. The filename is specified by the source attribute. If you have selected the Extension Type prior to this step the field will be pre-filled with the default value which you can change it if needed.

The attributes you can use to refer to installation folders are:
  • $Dreamweaver - Specifies the Dreamweaver installation folder
  • $Fireworks - Specifies the Fireworks installation folder
  • $Flash - Specifies the Macromedia Flash installation folder
  • $System - Specifies the System or System32 folder
  • $Fonts - Specifies the Font folder on the computer's hard disk
