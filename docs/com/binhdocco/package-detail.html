iPadFrameworkClasses
 


Main Classes
 ClassDescription
 ApplicationManager A singleton class to start up the framework.
 SceneObject Manages all screen objects: Add screen, Remove screen, Open PDF, Close PDF, Screen Transition.
 NavigationObject Manages the navigation: Show, Hide navigation, active/deactive subitem.
 Dispatcher Use to broadcast events through all classes.
 RestoreLastScreen Singleton class that manages shared object for saving information on iPad device. Also save the last screen index, so the presentation can restore to last screen it exited.
 NewScrollbar Class to create a scrollbar.
 AssetObjectContainer A subclass of MovieClip that contains many sub MovieClips in case of using vertical slide transition.
 AppEvent App Events such as: Open PDF, Open Mail, Save Image.
 NavigationEvent Navigation Events such as: Show, Hide, Change, Get active screen index.
 SceneEvent Scence Events such as: Enable/Disable changing screens.
 GotoSceneEvent Jumping Scence Events such as: Go to screen, Go to screen by offset (use for presentation has Vertical sliding), Close PDF .
 PersistenceDataEvent Tracking Events such as: Save tracking, Save survey.
